Abacus
Abacus is a general purpose discord bot meant for anyone! This website is meant as a place to get information about abacus, and to function as a control panel for server and user settings. Explore around a little bit! Abacus is meant to be as broad as possible, with highly customizable settings and functionality for nearly everything. It's in progress all the time, and suggestions are welcome! Join the support server and add Abacus below!
Features
  • Automoderation
  • Image manipulation
    • Testing hexadecimal colors
    • Testing discord profile pictures
    • Captioning images
    • Bitcrushing images
  • General use
    • Choosing between options
    • Flipping coins
    • Making polls
    • Calculating math
  • D&D
    • Integration with Foundry
    • Dice rolling
    • Rolling stats
    • Character management
  • Autoresponders
